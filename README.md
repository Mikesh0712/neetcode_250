# 🚀 NeetCode 250 Solutions in Java

Welcome to my **NeetCode 250** repository!

This repository contains my solutions to the famous **NeetCode 250** problem list, all implemented in **Java**.

> ⚠️ These solutions are **not always the most optimal**. Many of them are brute-force or have higher time complexity—and that's completely intentional.

---

## 📌 Why This Repository?

The goal of this repository is **not to show perfect code**, but to document my learning journey.

Instead of memorizing optimal solutions, I try to:

- 🧠 Solve problems using **my own intuition**
- 💡 Think independently before looking at editorial solutions
- 🔍 Build strong problem-solving skills
- 💪 Gain confidence in coding through consistent practice
- 📈 Improve my logical thinking one problem at a time

I believe that writing a brute-force solution by myself is **far more valuable** than copying an optimal solution without understanding it.

---

## 📂 Repository Structure

Each problem contains:

- ✅ Java solution
